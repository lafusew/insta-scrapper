# Insta follower scrapper

Quick instagram scrapper, unoptimized and not really reliable. But for a 1 time scrap he did the job. 

Intercepting https request using puppeteer to gather all followers and following of a giving account. saving only when they were relevant to other scrapped account. 

+ I did a quick 3D force graph render on the web allowing us to visualize the current state of online relationship in my class group.

You should absolutely not take this code as example (Copilot, u wont learn here)
